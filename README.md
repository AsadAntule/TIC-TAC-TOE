# TIC-TAC-TOE
This is a simple interactive Tic Tac Toe game created using HTML, CSS, and JavaScript.
The project focuses on CSS Grid layout, responsive design, and basic game logic.

# Features
•	Start, Game, and Game Over screens
•	3×3 Tic Tac Toe board using CSS Grid
•	Score tracking for X and O
•	Restart and Play Again options
•	Responsive layout
•	Hover effects and transitions

# Technologies Used
	•	HTML
	•	CSS (Grid & Flexbox)
	•	JavaScript

# How to Run
1.	Download or clone the repository
2.	Open index.html in a browser
3.	Click Start Game and play
